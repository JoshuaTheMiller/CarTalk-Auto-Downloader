# Running

Follow the steps below if you want to use this

## Prerequisites

* Install [node & npm](https://www.npmjs.com/get-npm)
* Install Typescript (`npm install -g typescript`)

## To run

1. `npm install`
2. `npm run start`

## To use CLI in development

1. `npm build`
2. `npm link` (only need to do this once)
